# Color Palette Generator

A simple and interactive color palette generator that allows users to create visually appealing color combinations instantly. Built using HTML, CSS, and JavaScript.

---

## Features

- Generate random color palettes  
- Copy hex color codes with a single click  
- Clean and minimal user interface  
- Responsive design for different screen sizes  
- Fast and lightweight implementation  

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## How It Works

- Click the "Generate" button to create a new color palette  
- Each color block displays its corresponding hex code  
- Click on a hex code to copy it to the clipboard  
- Use the generated palette for design or development purposes
